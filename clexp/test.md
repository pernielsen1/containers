# Bundesbank BISTA XML Test Facility

## Test File Submission

- Test files must have names beginning with `TEST_` (applies to both ZIP and XML filenames)
  - Example: `TEST_bista2411.xml`
- Submitted via the same portal as production — the `TEST_` prefix routes them as test data

## Onboarding / Go-Live Process

1. Register with ExtraNet or NExt
2. A mandatory test submission is required before going live
3. Go-live date is agreed bilaterally with Bundesbank after the test passes

## Relevant Portals

- [ExtraNet / NExt (Bankenstatistik)](https://www.bundesbank.de/de/service/meldewesen/bankenstatistik/extranet-next/bankenstatistik-extranet-next-611396) — main submission portal
- [NExt (Neukonzeption ExtraNet)](https://www.bundesbank.de/de/service/meldewesen/aussenwirtschaft/elektronische-einreichung/next-960348) — newer successor platform
- [ExtraNet general](https://www.bundesbank.de/de/service/extranet)

## Notes

- No separate open sandbox environment; testing happens within the production portal using the `TEST_` naming convention
